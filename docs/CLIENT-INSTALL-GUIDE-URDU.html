KitchenMart.pk POS

بھائی گی کروکری — انسٹال گائیڈ (کلائنٹ)
یہ صرف انسٹال پیکج ہے — سورس کوڈ شامل نہیں ہے۔
اس فولڈر کو کاپی کریں، START-POS.bat چلائیں، اور روزانہ کام شروع کریں۔

۱۔ کمپیوٹر کی ضرورت

ونڈوز Windows 10 یا Windows 11
ڈیٹا بیس SQL Server LocalDB (عام طور پر پہلے سے ہوتا ہے) یا SQL Express
.NET Runtime .NET 10 Desktop Runtime — اگر پروگرام نہ کھلے تو انسٹال کریں
https://dotnet.microsoft.com/download/dotnet/10.0
انٹرنیٹ ضروری نہیں — بلنگ اور انوینٹری آف لائن چلتی ہے

۲۔ پہلی دفعہ انسٹال

  1. یہ پورا فولڈر (یا ZIP) دکان کے کمپیوٹر پر کاپی کریں، مثلاً: C:\KitchenMart-POS
  2. POS فولڈر کھولیں
  3. START-POS.bat پر ڈبل کلک کریں
  4. پہلی دفعہ ڈیٹا بیس خود بن جائے گی — تھوڑا وقت لگ سکتا ہے
  5. لاگ ان اسکرین آ جائے تو کامیاب انسٹال ہے
نوٹ: اگر ونڈوز SmartScreen وارننگ دے تو More info → Run anyway منتخب کریں — یہ آپ کے POS سافٹ ویئر کا معمولی پیغام ہے۔

۳۔ لاگ ان

مالک (Owner) صارف نام: owner
پاس ورڈ: ڈیولپر الگ سلپ پر دے گا
سیلز (Sales) صارف نام: sales
پاس ورڈ: ڈیولپر الگ سلپ پر دے گا
پہلی دفعہ لاگ ان کے بعد Settings سے پاس ورڈ ضرور بدل لیں۔

۴۔ روزانہ استعمال

  • صبح: POS\START-POS.bat سے پروگرام کھولیں
  • بلنگ، واپسی (Returns)، اور اسٹاک (Inventory) یہیں سے کریں
  • شام کو پروگرام معمولی طریقے سے بند کریں
  • پورا فولڈر ڈیلیٹ نہ کریں — ڈیٹا اور سیٹنگز یہیں محفوظ ہیں

۵۔ رسید پرنٹر (اختیاری)

  1. ونڈوز میں پرنٹر انسٹال کریں
  2. POS میں Settings → Shop Settings کھولیں
  3. Hardware میں پرنٹر کا بالکل وہی نام لکھیں جو ونڈوز میں نظر آتا ہے
  4. محفوظ کریں اور ایک ٹیسٹ بل پر پرنٹ چیک کریں

۶۔ کلاؤڈ ڈیش بورڈ (اختیاری)

اگر موبائل یا دوسرے PC سے رپورٹ دیکھنی ہو:

  1. CLOUD-API\START-CLOUD-API.bat چلائیں
  2. براؤزر میں کھولیں: http://localhost:5050
  3. POS Settings میں Cloud Sync آن کریں (اگر ڈیولپر نے سیٹ کیا ہو)
کلاؤڈ بند رہے تو بھی POS بلنگ مکمل کام کرتی ہے۔

۷۔ بیک اپ

  • ہفتے میں ایک بار ڈیولپر سے SQL بیک اپ لینے کا طریقہ پوچھ لیں
  • پورا KitchenMart-POS فولڈر USB پر کاپی رکھنا بھی اچھا ہے
  • سیٹنگز کی فائل: POS\appsettings.json — محفوظ رکھیں

۸۔ عام مسائل

پروگرام نہیں کھلتا .NET 10 Desktop Runtime انسٹال کریں
ڈیٹا بیس ایرر SQL LocalDB / Express دوبارہ انسٹال کریں؛ ڈیولپر سے رابطہ کریں
پرنٹر نہیں چلتا ونڈوز پرنٹر ٹیسٹ پرنٹ کامیاب ہو؛ Settings میں نام درست لکھیں
5050 پر کلاؤڈ نہیں پہلے START-CLOUD-API.bat چلائیں
