Note: Existing shop data (products, sales, customers) rakhen — client ne verify kar liya hai. Is checklist mein sirf woh kaam hain jo delivery / install / handover ke liye zaroori hain.

Phase 1 — Release build banana (apne PC par)

  • POS app Release publish: dotnet publish src\ShopPOS.WPF\ShopPOS.WPF.csproj -c Release -o publish\ClientPOS
  • Cloud API Release publish (optional): dotnet publish src\ShopPOS.Api\ShopPOS.Api.csproj -c Release -o publish\ClientCloudApi
  • Publish folder mein ShopPOS.WPF.exe run karke final test
  • Debug build client ko na den — sirf Release / publish folder
  • Source code dena hai ya sirf EXE — client se clear karo aur note karo

Phase 2 — appsettings.json client-ready

  • ConnectionStrings — client PC ke SQL Server / LocalDB ke mutabiq set
  • Hardware:LogoPath — client logo ka sahi path (OneDrive / dev path hatao)
  • Hardware:ThermalPrinterName — shop printer ka naam (agar thermal hai)
  • CloudSync:ApiBaseUrl — localhost ya client server URL
  • CloudSync:ApiKey — strong naya key (dev key client ko mat do)
  • CloudSync:ShopId — kitchenmart-sargodha verify
  • SMS / WhatsApp / Email — disabled rakho jab tak client credentials na de

Phase 3 — Security (zaroori)

  • Owner login: owner — password owner123 se badal kar naya set karo
  • Sales login: sales — password sales123 se badal kar naya set karo
  • Settings → Change Password (owner account)
  • Settings → Security Questions save karwao (Forgot Password ke liye)
  • Naye passwords alag sealed note / envelope mein client ko do
  • Default passwords (owner123 / sales123) client ko kabhi mat batao written slip par

Phase 4 — Shop settings verify

  • Shop Name — Bhai Gee Crockery Store / KitchenMart.pk
  • Address — Karkhana Bazar, Sargodha
  • Phone — shop number sahi
  • Currency — Rs.
  • Tax rate — client ke mutabiq
  • Receipt footer message — thank you / return policy text
  • Settings → Save Shop Settings → “Settings Saved” popup aaye

Phase 5 — Full functional test (client machine jaisa)

  • Owner login + salesman login dono test
  • Billing — search, cart, discount, payment, receipt
  • Customer mobile / email (optional fields) test
  • Receipt print (printer connected ho to)
  • Inventory — product edit + save (tracking error fix verify)
  • Returns — original price par refund
  • Sales Report — aaj ki sales dikhe
  • Dashboard — totals sahi
  • Audit Logs — actions log ho rahe hon
  • App band karke dubara kholo — data save hai
  • PC restart ke baad app + data OK

Phase 6 — Cloud sync (optional — agar dena ho)

  • ShopPOS.Api (ClientCloudApi) client PC par copy
  • API startup shortcut / task scheduler (shop khulte hi chale)
  • Settings → Cloud Sync ON → URL + API Key + Shop ID save
  • Sync Now — “Synced X sale(s)…” message
  • Browser: http://localhost:5050 — dashboard sales/stock dikhe
  • Phone same WiFi: http://[PC-IP]:5050 test (optional)
  • Client ko samjhao: API band ho to sync ruke, local data safe rahega
